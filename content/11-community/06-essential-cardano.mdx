[Essential Cardano](https://github.com/input-output-hk/essential-cardano) is a central source for all things Cardano-related. 
This is an open source repository built for the community where we have outlined all the organizations, partners, and groups that are part of the Cardano ecosystem, as well as a comprehensive collection of resources to help you learn more about Cardano. 
Help us grow and expand this list by submitting a pull request with your contribution.
